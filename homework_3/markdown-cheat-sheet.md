# Отчет по лабораторной работе №3
# Проект: Предсказание вероятности позитивной метки с помощью градиентного бустинга на решающих деревьях

## Описание проекта
Данный проект направлен на разработку модели машинного обучения для бинарной классификации — предсказания вероятности принадлежности объекта к позитивному классу. В качестве основы используется градиентный бустинг, реализованный через библиотеку **LightGBM**.

---

## Основные этапы работы

### 1. Загрузка и подготовка данных
- Загружаются тренировочные и тестовые данные из CSV-файлов.
- Определяется целевая переменная `y`.
- Выбираются категориальные признаки для дальнейшей кодировки.

### 2. Обработка категориальных признаков
- Используется `OrdinalEncoder` из `sklearn` с параметрами:
  - `handle_unknown='use_encoded_value'` — для обработки новых категорий в тестовых данных.
  - `unknown_value=-1` — значение для неизвестных категорий.
- Кодировка обучается на тренировочных данных и применяется к тестовым.

### 3. Разделение данных
- Тренировочный набор делится на обучающую и валидационную выборки с помощью `train_test_split` для оценки качества модели.

### 4. Обеспечение соответствия признаков
- Проверяется наличие недостающих признаков в тестовых данных и добавляются с нулями для совпадения структуры данных.

### 5. Обучение модели
- Используется `LGBMClassifier` (LightGBM).
- Параметры модели:
  - `n_estimators=1000`
  - `learning_rate=0.05`
  - `class_weight='balanced'`
  - `random_state=42`
- Обучение проводится с ранней остановкой (`early_stopping`) для предотвращения переобучения.

### 6. Оценка модели
- ROC-AUC рассчитывается на валидционной выборке для оценки качества.

### 7. Предсказание и создание файла
- Генерация предсказаний вероятностей для тестовой выборки.
- Создается файл `submission.csv` с колонками `id` и `y`.
